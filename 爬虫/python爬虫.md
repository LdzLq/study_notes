### http基础
##### URL 统一资源定位符
1. URL标准格式
```
scheme://[username:password@]hostname[:port][/path][;parameters][?query][#fragment]
- scheme: 协议；如https、http、ftp等
- username:password@: 用户名和密码
- hostname: 主机名，可以是域名或IP地址
- port: 端口号，http默认80，https默认443
- path: 路径，网络资源在服务器中指定位置
- parameters: 参数，用来访问某个资源时的附加信息
- query: 查询某类资源，多个查询用&分隔
- fragment: 片段，内部书签
```
